# About
A bare minimum hyprland plugin to unfocus windows when the cursor leave it \
<video src="https://github.com/user-attachments/assets/b75cc866-f642-4574-b9c6-c3f9d44b1349" width="100%" controls></video>

# Setup
Add the repository if not already
```sh
hyprpm add https://github.com/hxero/hyplugs
```
then
```sh
hyprpm enable curleave
```
